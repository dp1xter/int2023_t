# New big-integer type int2023_t
This integer data type supports numbers in the range [-2^2022 + 1; 2^2022 - 1].  
Storage in **256-number-system.**

# Operations
The following operations are defined for it:  
- "+" - *addition.*
- "-" - *subtraction.*
- "*" - *multiple.*
- "/" - *division.*
- "==" / "!=" - *comparison.*
- "<<" - *output in 256-number-system.*
